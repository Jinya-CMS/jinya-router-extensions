# Introduction

Official extensions for the Jinya Router.

## Getting Started

To get started with Jinya Router Extensions, you can install it via composer:

```bash
composer require jinya/router-extensions
```

## Features

- Database integration for Jinya Router
- Automatic route generation based on entity attributes
- JSON and XML support out of the box
- Built-in validation for required fields
- PSR-7 and PSR-15 compatible
